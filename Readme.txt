# Soccer Match Prediction System

## Description
This repository contains a comprehensive soccer match prediction system. It includes data loading, preparation, exploratory data analysis (EDA), and visualization of relationships between first-half and total goals. The project leverages Python libraries such as pandas, seaborn, matplotlib, and statsmodels to analyze and visualize soccer match data across multiple seasons.

## Installation
1. Clone the repository:
git clone https://github.com/your-username/soccer-match-prediction-system.git

2. Navigate to the project directory:
cd soccer-match-prediction-system

3. Install the required dependencies:
pip install -r requirements.txt


## Usage
1. Ensure you have the necessary CSV files in the specified directory:
C:/Users/34633/OneDrive/Desktop/soccer match prediction system/matches_system_overview/season_fixed_shorted/

2. Run the main script:
python main.py


## Project Structure
- `main.py`: The main script to run the project.
- `requirements.txt`: List of dependencies required for the project.
- `data/`: Directory containing the CSV files for different seasons.

## Features
- Data loading and preparation
- Exploratory Data Analysis (EDA)
- Visualization of relationships between first-half and total goals
